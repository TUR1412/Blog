# 轩天帝 · 轩少｜修仙纪事（个人博客）

一卷“像真事”的修仙题材个人站：不写夸饰，不写旁门；只记“轩天帝（轩少）”的行止、分寸、规矩与年表。

## 亮点

- **世界级视觉底座**：极光氛围 + 玻璃拟态 + 细噪点纹理，拒绝死白/死黑。
- **世界级交互手感**：全站 Q 弹点击反馈；收藏、展开、筛选都有即时响应。
- **灵镜检索**：按 `Ctrl/⌘ + K`（或 `/`）召出命令面板，直达页面/篇章。
- **收藏与续读定位**：单篇可收藏；收藏页支持拖拽排序/取消收藏/导入导出；阅读会记住停在的章节与段落，随时续读；支持沉浸阅读。
- **洞府图（年表可视化）**：把年表写成“路”，可搜索、可 ↑↓ 走位选点；支持“洞府四层”筛选（立身/立名/立规/立秤）与气象筛选（平/明/警）；按节点直达对应纪事，一键收入札记。
- **路标批注（洞府图）**：每个节点可写一句短评（自动保存），并可一键并入札记；支持导入/导出存档（.json）与批注文卷导出（.txt）；并提供“批注总览”（筛选/检索/定位/并入札记），便于复盘与归档。
- **关系谱（对照之网）**：把“人、誓词、旧物、关口与地点”连成一张谱；点任一节点可见牵连，一键收入札记；支持筛选/检索，并提供“只看牵连”聚焦视图。
- **关系谱节点批注**：每个节点可写一句“对照”（自动保存），并提供“批注总览”（检索/定位/并入札记），把线索写成能复盘的路。
- **札记（本地持久化）**：输入即保存，刷新不丢；支持复制、导出文卷（.txt）/存档（.json）、导入文卷/存档。
- **藏品拖拽排序**：拖一拖就能换位，排序会记住。

## 开始

安装依赖：

```bash
npm install
```

本地预览（会启动开发服务，按需使用）：

```bash
npm run dev
```

构建静态产物：

```bash
npm run build
```

构建后预览：

```bash
npm run preview
```

## 约束

- 站点正文只围绕“修仙纪事与人物志”，避免无关内容。
- 用户生成内容默认写入本地存储（`localStorage`），防止“刷新即丢”的幽灵问题。
