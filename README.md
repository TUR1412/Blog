# 轩天帝 · 轩少｜修仙纪事（个人博客）

一卷“像真事”的修仙题材个人站：不写夸饰，不写旁门；只记“轩天帝（轩少）”的行止、分寸、规矩与年表。

## 亮点

- **世界级视觉底座**：极光氛围 + 玻璃拟态 + 细噪点纹理，拒绝死白/死黑。
- **世界级交互手感**：全站 Q 弹点击反馈；收藏、展开、筛选都有即时响应。
- **灵镜检索**：按 `Ctrl/⌘ + K`（或 `/`）召出命令面板，直达页面/篇章。
- **札记（本地持久化）**：输入即保存，刷新不丢；支持导出与复制。
- **藏品拖拽排序**：拖一拖就能换位，排序会记住。

## 开始

安装依赖：

```bash
npm install
```

本地预览（会启动开发服务，按需使用）：

```bash
npm run dev
```

构建静态产物：

```bash
npm run build
```

构建后预览：

```bash
npm run preview
```

## 约束

- 站点正文只围绕“修仙纪事与人物志”，避免无关内容。
- 用户生成内容默认写入本地存储（`localStorage`），防止“刷新即丢”的幽灵问题。
