# 轩天帝 · 轩少｜修仙纪事（个人博客）

一卷“像真事”的修仙题材个人站：不写夸饰，不写旁门；只记“轩天帝（轩少）”的行止、分寸、规矩与年表。

## 亮点

- **世界级视觉底座**：极光氛围 + 玻璃拟态 + 细噪点纹理，拒绝死白/死黑。
- **世界级交互手感**：全站 Q 弹点击反馈；收藏、展开、筛选都有即时响应。
- **灵镜检索**：按 `Ctrl/⌘ + K`（或 `/`）召出命令面板，直达页面/篇章。
- **收藏与续读定位**：单篇可收藏；收藏页支持拖拽排序/取消收藏/导入导出；阅读会记住停在的章节与段落，随时续读；支持沉浸阅读。
- **洞府图（年表可视化）**：把年表写成“路”，可搜索、可 ↑↓ 走位选点；支持“洞府四层”筛选（立身/立名/立规/立秤）与气象筛选（平/明/警）；按节点直达对应纪事，收入札记可选连同批注。
- **路标批注（洞府图）**：每个节点可写一句短评（自动保存），批注编辑区支持“卷内检索”（上一处/下一处跳转）；并可一键并入札记；支持导入/导出存档（.json）与批注文卷导出（.txt）；并提供“批注总览”（筛选/检索/定位/并入札记），便于复盘与归档；另有“收入札记（含批注）”一键合并。
- **关系谱（对照之网）**：把“人、誓词、旧物、关口与地点”连成一张谱；点任一节点可见牵连，一键收入札记；支持筛选/检索，并提供“只看牵连”聚焦视图。
- **关系谱节点批注**：每个节点可写一句“对照”（自动保存），批注编辑区支持“卷内检索”（上一处/下一处跳转）；并提供“批注总览”（检索/定位/并入札记）；支持导出文卷（.txt）/导出存档（.json）与导入存档，便于备份迁移；另有“收入札记（含批注）”一键合并。
- **批注馆（全站汇总）**：把洞府图与关系谱的批注收拢到一处；可检索/筛选/导出文卷/导入存档；支持“卷内检索”（命中高亮 + 上一处/下一处跳转）；导入前会显示差异预览（回声刻印）以避免误覆盖；导入/清心前会自动落下“护卷符快照”，可一键下载备份；支持同源联动（直达洞府/关系/纪事）与“封印整卷”；另有“清心诀”用于剔除空批注/按源清空，并支持回退撤销。
- **札记（本地持久化）**：输入即保存，刷新不丢；支持“经卷观”（Markdown 排版预览，自动把 `【标题】` 行转为标题）与“经卷目录”（标题定位 + URL 分享 + 目录折叠/检索 + 一键复制定位）以及“章节内检索”（命中高亮 + 上一处/下一处），并支持复制、导出文卷（.txt）/存档（.json）、导入文卷/存档。
- **藏品拖拽排序**：拖一拖就能换位，排序会记住。

## 开始

安装依赖：

```bash
npm install
```

本地预览（会启动开发服务，按需使用）：

```bash
npm run dev
```

构建静态产物：

```bash
npm run build
```

构建后预览：

```bash
npm run preview
```

## 部署到 GitHub Pages

本仓库已内置 GitHub Actions 工作流：推送到 `main` 会自动 `npm ci` + `npm run build`，并把 `dist/` 发布到 GitHub Pages。

1. 在 GitHub 仓库打开：Settings → Pages
2. Source 选择：GitHub Actions
3. 等待 Actions 跑完后，访问：`https://<username>.github.io/Blog/`

## 约束

- 站点正文只围绕“修仙纪事与人物志”，避免无关内容。
- 用户生成内容默认写入本地存储（`localStorage`），防止“刷新即丢”的幽灵问题。
