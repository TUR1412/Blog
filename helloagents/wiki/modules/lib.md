# lib

## 目的
提供跨模块可复用的工具能力（存储、查找高亮、性能采样、触感反馈等）。

## 模块概述
- **职责:** cn、storage、find、perf、haptics、focus 等
- **状态:** ✅稳定
- **最后更新:** 2025-12-31

## 核心能力
- `focus.ts`：提供可复用的焦点收敛（focus trap）工具，供 Overlay/命令面板等弹层类 UI 统一使用

## 规范
### 需求: 外部输入必须兜底
**模块:** lib
对 URL / 导入JSON 等外部输入必须进行校验、收敛、降级处理，避免运行时异常或安全风险。

#### 场景: 导入JSON
导入文件缺字段/版本不匹配/内容损坏时，不应导致页面崩溃。
- 预期结果：给出友好提示并保持现有数据不被污染
